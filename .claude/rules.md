# Regras de Qualidade

## 1. Testes são obrigatórios

Todo novo código de domínio ou regra de negócio deve ser acompanhado de testes automatizados.

- Backend: testes unitários obrigatórios para regras de negócio.
- Frontend: testes para componentes com lógica (não apenas renderização estática).
- Não é permitido implementar regra de negócio sem cobertura mínima de testes.
